    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="user-content-android-click" class="anchor" href="#android-click" aria-hidden="true"><span class="octicon octicon-link"></span></a>android-click</h1>

<p>Click game for android</p>

<p>A simple game where a user must click a button as many times as possible in 10 seconds.</p>

<p>User can select new game, high score, or quit.</p>

<p>New game presents the user with the game screen. The game begins when the user clicks the first time.
When the time runs out, the user is presented with a dialog that displays their score, the current high score and how many clicks per second they achieved.
The dialog provides options to play a new game or go b ack to the main screen.</p>

<p>High score presents the user with a dialog displaying the current high score.</p>

<p>Quit exits the app.</p></article>
